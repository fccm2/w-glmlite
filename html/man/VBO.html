type bo_param =
| GL_BUFFER_ACCESS
| GL_BUFFER_MAPPED
| GL_BUFFER_SIZE
| GL_BUFFER_USAGE

params for glGetBufferParameter, but not used

in the C API glGetBufferParameteriv can be called with GL_BUFFER_ACCESS, GL_BUFFER_MAPPED, GL_BUFFER_SIZE, or GL_BUFFER_USAGE, each of these parameters return a different type, so glGetBufferParameter have been wrapped as follow:

  • glGetBufferParameter GL_BUFFER_ACCESS => glGetBufferAccess
  • glGetBufferParameter GL_BUFFER_MAPPED => glGetBufferMapped
  • glGetBufferParameter GL_BUFFER_SIZE => glGetBufferSize
  • glGetBufferParameter GL_BUFFER_USAGE => glGetBufferUsage
